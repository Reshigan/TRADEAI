# Seed Data

This directory contains generated test data for Trade AI platform.

## Files Generated
- customers.json (1000+ customers)
- products.json (500+ products)  
- transactions.json (50k+ transactions)

## Generation
Run: `node scripts/seed-comprehensive-data.js`

## Usage
Import this data for:
- AI/ML model training
- Performance testing
- Demo scenarios
- Analytics validation

## Data Quality
- Realistic business scenarios
- Seasonal patterns
- Multiple customer types
- Product categories
- Promotional activities
