# SAP Import Templates for TRADEAI

## Overview
This package contains CSV templates for importing SAP data into TRADEAI's FMCG Trade Spend Management platform.

## Templates Included

1. **customers.csv** - Customer master data (SAP KNA1/KNVV)
2. **products.csv** - Material master data (SAP MARA/MARC)
3. **sales_actuals.csv** - Billing documents (SAP VBRK/VBRP)
4. **deductions.csv** - AR deductions (SAP FI-AR)
5. **trading_terms.csv** - Trading term agreements
6. **promotions.csv** - Planned promotions

## Quick Start

1. Download the appropriate template
2. Fill in your SAP data
3. Upload via TRADEAI Import Center
4. Verify import results

## Documentation

See `docs/SAP_IMPORT_GUIDE.md` for complete field definitions, validation rules, and troubleshooting.

These templates are compatible with Power BI for reporting and analytics. Export data from TRADEAI in the same format for seamless integration.
